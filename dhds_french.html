Général

Dans la série, se trouvent 5 types d'appareil de composition :

  1. Voie lactée
  2. Pégase
  3. Atlantis
  4. Jumper
  5. Le controleur d'Universe

Les DHD sont les premières manières de composition d'une Stargate dans la série, ils se connectent aux Stargates proches avec une connexion en sous-espace. Dans les DHD terrestes se trouve un réacteur à fusion froide conçu pour durer des milliers d’années pour alimenter la porte connectée

Le controleur d'Universe

Le controleur d'Universe n'est pas un DHD, dans la série il communique avec la rampe d'embarquement de la Stargate d'Universe la plus proche. Dans le mod au lieu de communiquer avec la rampe le contrôleur d’Univers communique directement avec la porte. Le controleur d'Universe possède 4 modes :

  1. Proximité
  2. Mémoire
  3. Anneaux
  4. OC

Le mode à proximité affiche et vous permet de composer les adresses de l’univers de toutes les portes dans 1024 blocs (configurable). Le mode Mémoire vous permet de composer n’importe quelle adresse que vous avez enregistrée dans votre controleur de la même manière que vous enregistrez une adresse sur une page. Le mode anneaux vous permet de contrôler l’ensemble d’anneaux de transport le plus proche de votre emplacement. Le mode OC vous permet de diffuser un message OC sans fil vers un port spécifique à capter soit par une porte pour être relayé à travers un vortex ou être récupérer par un ordinateur, vous pouvez également spécifier une adresse de modem afin qu’il ne soit envoyé qu’à un ordinateur spécifique.

Voie lactée

Le DHD de la Voie lactée ne peux, par défaut, ne peux activer que 7 chevrons mais peux être améliorer avec un Cristaux de Symboles DHD pour être capable d'activer 9 chevrons. Le DHD possède 39 bouttons, dont 38 Symboles et le GBR (Gros Bouton Rouge) pour activer le vortex.

Pégase

Le DHD de Pégase ne peux, par défaut, ne peux activer que 7 chevrons mais peux être améliorer avec un Cristaux de Symboles DHD pour être capable d'activer 9 chevrons. Le DHD possède 39 bouttons, dont 36 Symboles utilisables, 2 Sumboles désactivées et le GBB/GBV (Gros Bouton Bleur/Vert) pour activer le vortex.

Atlantis

Le DHD d'Atlantis n'a pas encore été implémenté dans le mod.
